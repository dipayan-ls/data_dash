# Project Dash — Unified Rebuild Plan

## Background

The project at `/Users/dipayandhar/Pictures/project_dash` is a **Data Integrity Dashboard** that:
1. Runs Python scrapers against 9 ad platform APIs (Google, Meta, Microsoft, Pinterest, Snapchat, TikTok, Twitter/X, Amazon, Reddit)
2. Compares API data against BigQuery warehouse data
3. Displays discrepancies in a React dashboard with charts and tables

Currently the project is split across two nearly-identical folders (`combined/` and `data-integrity-dashboard/`), has competing backend implementations (`app.py` Flask vs `server.ts` Express), and has orphaned/duplicate scrapers sitting in the root.

**Goal**: Flatten into one unified project under `data-integrity-dashboard/` that can be run locally with two commands.

## Verification Plan

### Manual Verification Steps

**Step 1 — Install dependencies**
```bash
cd /Users/dipayandhar/Pictures/project_dash/data-integrity-dashboard
pip install -r requirements.txt
npm install
```

**Step 2 — Start the Flask backend**
```bash
python3 app.py
# Should print: Running on http://0.0.0.0:3000
```

**Step 3 — Start the Vite frontend (separate terminal)**
```bash
npm run dev
# Should print: Local: http://localhost:5173
```

**Step 4 — Open http://localhost:5173 in browser**
- Confirm the DataAudit dashboard loads with sidebar
- Click "Run Auto Scraper" — confirm the dialog opens and shows all 9 channels (not just 3)
- Confirm workspaces load from the backend API

**Step 5 — Test CSV Upload**
- Use one of the sample CSVs (e.g. from scraper output) to upload via "Choose File"
- Confirm the channel list in the sidebar populates
- Confirm charts and data table render
